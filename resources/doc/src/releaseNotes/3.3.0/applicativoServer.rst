Nuova funzionalità di registrazione di un Applicativo Server
--------------------------------------------------------------

Gli applicativi registrati su GovWay, dalla versione 3.3.0, possono essere di due tipi:

- *Client*: si tratta degli applicativi presenti anche nelle precedenti versioni, censiti allo scopo di identificarli ed autorizzarli durante l'invocazione di erogazioni o fruizioni di API;

- *Server*: questo nuovo tipo di applicativo consente di censire una applicazione di backend alla quale gli vengono associati i dati che tipicamente venivano indicati nella sezione 'Connettore' di una erogazione di API (endpoint, credenziali, ...). In una erogazione è quindi possibile riferire l'applicativo server precedentemente registrato come modalità alternativa a quella di indicare tutte le informazioni di backend.
