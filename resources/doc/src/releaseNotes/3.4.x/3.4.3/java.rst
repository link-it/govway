Java 25 e Librerie 3Parti
-------------------------------

Dalla versione 3.4.3, GovWay introduce il supporto alla versione 25 di Java, la più recente release LTS (Long-Term Support) disponibile al momento.
	
Sono state inoltre aggiornate alle versioni più recenti tutte le librerie terze parti utilizzate. Tra le più rilevanti la migrazione a Spring 7 e ai relativi API Jakarta EE 11 e la versione 4.2.0 di cxf per la gestione delle API Rest e Soap.
