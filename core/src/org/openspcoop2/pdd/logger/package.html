Fornisce la classe Logger che definisce un logger utilizzato per registrare su file e su db i tracciamenti e i messaggi diagnostici generati dalla porta di dominio. Le impostazioni del Logger sono settabili all'interno del file 'govway.log4j2.properties'. La classe MsgDiagnostico permette di emettere messaggi diagnostici secondo specifica. La classe Tracciamento permette di tracciare buste secondo specifica del protocollo
