Fornisce classi che implementano le modalita' di scambio messaggi tra i nodi funzionali dell'architettura di OpenSPCoop. Le classi devono implementare le interfacce INode*. Attuali implementazioni fornite sono per:
  • db: La gestione delle comunicazione avviene interamente attraverso un database relazionale
  • jms: Le comunicazioni sono mediate da code JMS. Un TransactionManager e' utilizzato per la gestione delle transazioni tra jms e db.
